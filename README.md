# kum0nga
A simple joomla scan

## Usage:

perl kum0nga.pl --url=URL [OPTIONS]

Options:

--user-agent => Use a custom user-agent header

--version    => Show joomla version

--enum-vul   => Enumerate components with known vulnerabilities
